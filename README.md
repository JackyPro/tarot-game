# 🔮 AI 塔羅之眼 (AI Tarot Eye)

這是一個基於 **PyScript** 與 **Gemini API** 開發的網頁占卜遊戲。

## 🚀 功能特點

- **自動抽牌**：隨機抽取三張大阿爾克那，支援正逆位。
- **雙模式解牌**：
  - 本地模式：不消耗 Token，快速獲取關鍵字。
  - (Not Support Yet) AI 模式：連接 Gemini 1.5 Flash 進行深度占卜解讀。
- **部署環境**：利用 GitHub Actions 自動部署至 GitHub Pages。

## 🛠️ 開發環境

- Docker + VS Code Dev Containers
- Python 3.12 + PyScript
- Gemini CLI
